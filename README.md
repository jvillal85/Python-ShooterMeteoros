# Python-ShooterMeteoros

Shooter Meteoros es un juego desarrollado en Python utilizando la biblioteca Pygame. En este juego, el jugador controla una nave espacial que debe disparar y destruir meteoros para ganar puntos. El juego incluye gr√°ficos, sonidos y animaciones para una experiencia de juego completa.

## Instrucciones

- **Mover Izquierda:** Presiona la tecla `<`
- **Mover Derecha:** Presiona la tecla `>`
- **Disparar:** Presiona la tecla `SPACEBAR`
